# 01 Introduction

## What's KrabbaScript?
As the README.md said, KrabbaScript is a statically typed, AOT compiled, high level programming language designed to be easy to write and read.

## Some history
KrabbaScript was designed to be a superior version of C/C++ to write some general purpose programs and designed for the embedded world.

## Who's KrabbaScript for?
KrabbaScript is for people that want the following:

- Readable code
- Speed of C
- Easy to prototype with

## Foreword
Before you start, this "documentation" assumes you already have some knowledge on writing C programs.
